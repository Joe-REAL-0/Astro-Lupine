# 星际“狼”客 (Astro Lupine) - 状态效果设计文档 (Buffs & Debuffs)

本文档记录了星际“狼”客 Mod 中特有的状态效果，包括赋予敌人的负面状态以及赋予玩家的增益/负面状态。

## 敌人 Debuffs (附加到敌人身上)

| 状态名称 | 效果描述 | 层数机制 |
| :--- | :--- | :--- |
| **木马病毒** | 玩家的回合开始时，该敌人受到相当于玩家【攻击寄存器】数值的伤害。 | 层数表示持续的回合数。每回合结束时层数减1。 |
| **零日漏洞** | 无法获取护甲（格挡）。 | 层数表示持续的回合数。每回合结束时层数减1。 |

---

## 玩家 Buffs (附加到玩家身上)

| 状态名称 | 效果描述 | 层数机制 |
| :--- | :--- | :--- |
| **自动维护** | 每回合开始时，获得相当于当前【防御寄存器】数值的格挡。 | 层数表示持续的回合数。每回合结束时层数减1。 |
| **攻击累加器** | 每回合开始时，【攻击寄存器】数值自增 `x`，`x` 为该Buff的层数。 | 层数 `x` 决定了每次增加的数值量。 |
| **防御累加器** | 每回合开始时，【防御寄存器】数值自增 `x`，`x` 为该Buff的层数。 | 层数 `x` 决定了每次增加的数值量。 |
| **内核加固** | 所有寄存器都被锁定，无法写入，只能被读取。 | 层数表示持续的回合数。每回合结束时层数减1。 |

---

## 玩家 Debuffs (附加到玩家身上)

| 状态名称 | 效果描述 | 层数机制 |
| :--- | :--- | :--- |
| **攻击覆写** | 下回合开始时，将【攻击寄存器】的数值强制设置为 `x`，`x` 为该Debuff的层数。 | 层数 `x` 即为覆写的目标数值，生效后该状态移除。 |
| **防御覆写** | 下回合开始时，将【防御寄存器】的数值强制设置为 `x`，`x` 为该Debuff的层数。 | 层数 `x` 即为覆写的目标数值，生效后该状态移除。 |
| **抽牌覆写** | 下回合开始时，将【抽牌寄存器】的数值强制设置为 `x`，`x` 为该Debuff的层数。 | 层数 `x` 即为覆写的目标数值，生效后该状态移除。 |
