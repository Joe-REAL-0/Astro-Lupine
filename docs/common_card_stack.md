### 一、 攻击牌

**重写重击**

* **费用：** 3
* **基础效果：** 造成 18 点伤害。**【写入】**。
* **升级效果：** 造成 22 点伤害。**【写入】**。
* **设计思路：** 基础的大型数值源。根据非对称设计，攻击寄存器允许有较高的爆发上限。打出后不仅造成高伤，还会将寄存器拉高至10（升级12），为后续的读取牌铺路。

**叠势**

* **费用：** 1
* **基础效果：** 造成 6 点伤害。**【读取】**，**【写入】**。
* **升级效果：** 造成 8 点伤害。**【读取】**，**【写入】**。
* **设计思路：** 核心成长牌。打出时先读取（6+4=10），然后写入10。下一次打出就变成（10+6=16）并写入16，实现左脚踩右脚的无限成长。

**兔子炸弹**

* **费用：** 0
* **基础效果：** 造成 0 点伤害，将此牌的一张复制加入你的弃牌堆。**【读取】**
* **升级效果：** 造成 2 点伤害，将此牌的一张复制加入你的弃牌堆。**【读取】**

**多线程**

* **费用：** 2
* **基础效果：** 造成 0 点伤害 2 次。**【读取】**。
* **升级效果：** 造成 0 点伤害 3 次。**【读取】**。
* **设计思路：** 完美利用独立加成的多段攻击。在默认情况下是1费6×2=12点伤害，一旦寄存器养起来，多段读取的收益极高。

**广域打击**

* **费用：** 1
* **基础效果：** 对所有敌人造成 0 点伤害，**读取**，给予 1 层 **木马病毒**。
* **升级效果：** 对所有敌人造成 0 点伤害，**读取**，给予 2 层 **木马病毒**。
* **设计思路：** 基础的AOE牌。同时可以提供一层木马病毒，下回合对敌人继续造成伤害。

**拟态：铁斩波**

* **费用：** 1
* **基础效果：** 造成0点伤害，获得0点格挡。**【读取】**。
* **升级效果：** 造成2点伤害，获得2点格挡。**【读取】**。
* **设计思路：** 能打能防，比较全面的卡牌。

**算力超载**

* **费用：** 1
* **基础效果：** 造成 12 点伤害，获得 抽牌覆写 1。
* **升级效果：** 造成 15 点伤害，获得 抽牌覆写 1。
* **设计思路：** 提供高额直接伤害与攻击写入（10或12），代价是将抽牌寄存器重置为1（抽牌能力降低）。适合在不缺手牌但急需输出终端爆发的回合使用。

**融断机制**

* **费用：** 1
* **基础效果：** 给予目标2层**零日漏洞**，造成9伤害，
* **升级效果：** 给予目标2层**零日漏洞**，造成12伤害
* **设计思路：** 稳定触发病毒效果的卡牌，基础伤害和升级伤害都很不错，让使用者能稳定的触发病毒效果。

**病变**

* **费用：** 1
* **基础效果：** 造成0点伤害 **读取** 敌人身上每有一层木马病毒，就额外造成一次伤害。
* **升级效果：** 造成2点伤害 **读取** 敌人身上每有一层木马病毒，就额外造成一次伤害。
* **设计思路：** 利用病毒层数造成伤害。

### 二、 技能牌

**防火墙**

* **费用：** 1
* **基础效果：** 获得 8 点格挡。**【写入】**。
* **升级效果：** 获得 12 点格挡。**【写入】**。
* **设计思路：** 由于防御寄存器的非对称设计，纯防御牌被严格限制上限。它能提供稳定的基础格挡和写入（6或8），但无法直接叠出极高的面板。

**堆栈防护**

* **费用：** 0
* **基础效果：** 获得 1 点格挡。**【读取】**，**【写入】**。
* **升级效果：** 获得 2 点格挡。**【读取】**，**【写入】**。
* **设计思路：** 防守端的成长牌。先读取（1+5=6），然后写入6。让你的防线随着战斗进行越来越厚。

**卸载**

* **费用：** 0
* **基础效果：** 获得 2 能量，获得 防御覆写 5。
* **升级效果：** 获得 2 能量，获得 防御覆写 8。
* **设计思路：** 0费白嫖格挡与费用，但代价是会把防御寄存器“降维”。需要在防守压力小或者实在缺费时斟酌使用，考验玩家对状态机的控制。

**销毁模块**

* **费用：** 1
* **基础效果：** 消耗 1 张牌，获得 8 点格挡。**【读取】**
* **升级效果：** 消耗 1 张牌，获得 12 点格挡。**【读取】**
* **设计思路：** 滤牌与防守的结合。配合基础的防御寄存器，能提供非常可观的格挡收益。

**缓存提取**

* **费用：** 1
* **基础效果：** 抽 1 张牌。**【读取】**。
* **升级效果：** 抽 2 张牌。**【读取】**。
* **设计思路：** 经典的过牌。默认读取2，相当于1费抽2。升级后1费抽3。

**预加载**

* **费用：** 0
* **基础效果：** 抽 2 张牌。获得攻击覆写 8。
* **升级效果：** 抽 2 张牌。获得攻击覆写 12。
* **设计思路：** 0费白嫖抽卡，但是会强制覆盖攻击寄存器

**覆写**

* **费用：** 0
* **基础效果：** 技能。选择手牌中的一张牌，为其添加【写入】。**消耗**。
* **升级效果：** 技能。选择手牌中的一张牌，为其添加【写入】。**消耗** **保留**。
* **设计思路：** 核心“跳板”。当你打出了一张吃到了其他加成的超高伤害牌或高防牌，用这张牌给它打上写入补丁，瞬间将对应寄存器拉满。

**接口拓展**

* **费用：** 1
* **基础效果：** 技能。从手牌中选择一张牌，使其同时获得 **【写入】** 和 **【消耗】**。
* **升级效果：** 费用降为 0 费。*(效果不变)*
* **设计思路：** 成长启动器。相当于强行造一张一次性覆盖高数值的写入牌

**汇集**

* **费用：** 3
* **基础效果：** 技能。获得 5 点能量。**虚无**。
* **升级效果：** 技能。获得 5 点能量。
* **设计思路：** 简单直接的爆发充能。寄存器流派的打法往往是“先打高费写入，再打低费读取”，前期极其需要这种启动能量。

**调和函数**

* **费用：** 0
* **基础效果：** 技能。获得 1 点能量。抽 0 张牌。**【读取】** **消耗**。
* **升级效果：** 技能。获得 1 点能量。抽 0 张牌。**【读取】**。
* **设计思路：** 费用置换与滤牌工具。换2费并过牌，帮助你在一回合内完成更复杂的“写入-读取”联动操作。

**动能转化**

* **费用：** 1
* **基础效果：** 获得 14 点格挡，获得攻击覆写8。**【写入】**。
* **升级效果：** 获得 17 点格挡，获得攻击覆写8。**【写入】**。
* **设计思路：** 非对称设计的核心防守支点。当面临高压时，牺牲高额的攻击寄存器（强制重置为3），换取爆发式的格挡收益与防御写入。升级后增加保留，让玩家能够将这张“救命牌”一直捏在手里，在最完美的时机打出置换。
